## 3. 发现并修复的问题

### 3.1 阻塞问题（本次发现并修复）

**问题 1：后端 DocumentProcessor 不支持 .txt 文件**

- 位置：`backend/app/services/document_processor.py`
- 现象：上传 .txt 文件后 process_status 变为 "failed"，错误信息 "不支持的文件类型: text"
- 原因：`file_utils.py` 将 .txt 映射为 "text" 类型，但 DocumentProcessor 只处理 pdf/word/image
- 影响：**文本文件无法通过 API 触发 agent audit**（CLI 可以，因为 agent 有自己的 parser）
- 修复：新增 `_process_text` 方法，支持 UTF-8/GB18030/GBK 编码

**问题 2：AGENT_LLM_PROVIDER 默认值不一致**

- 位置：`agent/config.py:51` vs `backend/app/core/config.py:56`
- 现象：agent 默认 "siliconflow"，backend 默认 "mimo"，.env.example 建议 "mimo"
- 修复：agent/config.py 改为 "mimo"

### 3.2 非阻塞问题（本次修复）

**问题 3：dead import in report_writer.py**

- `from agent.tools.risk_matrix import format_risk_summary` 从未使用

**问题 4：dead import + 未使用调用 in risk_assessor.py**
