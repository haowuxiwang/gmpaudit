### 2. SQLite WAL 模式

**文件变更**:
- `backend/app/core/database.py` — 添加 PRAGMA 配置

**技术细节**:
```python
PRAGMA journal_mode=WAL      # 并发读写
PRAGMA busy_timeout=5000     # 等待 5s 再报 SQLITE_BUSY
PRAGMA synchronous=NORMAL    # 平衡持久性和性能
PRAGMA cache_size=-64000     # 64MB 缓存
PRAGMA foreign_keys=ON       # 启用外键约束
```
