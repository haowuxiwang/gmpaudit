## 8. 关键技术决策记录

| 决策 | 原因 |
|------|------|
| LightRAG 替代 Microsoft GraphRAG | 轻量、支持本地 embedding、中文友好 |
| 确定性路由替代 LLM 路由 | Supervisor 用状态位路由，不依赖 LLM 判断 |
| dev_user 自动创建替代登录 | 本地桌面工具，不需要多用户认证 |
| ModelScope 替代 HuggingFace | 国内下载更快更稳定 |
| asyncio.Lock 保护单例 | 防止并发创建多个 LightRAG 实例 |
| UTF-8 → gb18030 → gbk 三级降级 | 中文文档编码不统一，需要兼容 |
